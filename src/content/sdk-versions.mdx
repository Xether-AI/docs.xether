## Quick Start

### Python
```bash
pip install xether-ai
```

```python
from xether_ai import XetherClient

client = XetherClient(api_key="your-api-key")
```

### JavaScript/TypeScript
```bash
npm install @xether/sdk
```

```javascript
import { XetherClient } from '@xether/sdk';

const client = new XetherClient({ apiKey: 'your-api-key' });
```

### Go
```bash
go get github.com/xether-ai/go-sdk
```

```go
import "github.com/xether-ai/go-sdk"

client := xether.NewClient("your-api-key")
```

### Java
```xml
<dependency>
  <groupId>ai.xether</groupId>
  <artifactId>sdk</artifactId>
  <version>1.0.5</version>
</dependency>
```

```java
import ai.xether.XetherClient;

XetherClient client = new XetherClient("your-api-key");
```

## Version Policy

We follow [Semantic Versioning](https://semver.org/) for all SDKs:

- **Major versions** (X.0.0) - Breaking changes, requires migration
- **Minor versions** (0.X.0) - New features, backward compatible
- **Patch versions** (0.0.X) - Bug fixes, fully backward compatible

## Security

### Vulnerability Reporting
If you discover a security vulnerability in any SDK:

1. **Do not** open a public issue
2. **Email** security@xether.ai with details
3. **We'll respond** within 48 hours
4. **Patch** will be released according to severity

### Dependencies
All SDKs undergo regular security audits and dependency scanning. We update dependencies promptly when vulnerabilities are discovered.

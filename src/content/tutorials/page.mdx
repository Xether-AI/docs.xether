## Tutorial Environment Setup

### Required Tools
- **Xether AI Account**: [Sign up](https://app.xether.ai/signup)
- **Python SDK**: `pip install xether`
- **CLI Tool**: `pip install xether-cli`
- **Docker** (optional): For containerized pipelines

### Sample Data
All tutorials use sample datasets that are:
- Realistic and representative
- Small enough for quick processing
- Available in the tutorial repositories
- Well-documented with schemas

### Development Environment
Set up your environment:
```bash
# Install required packages
pip install xether xether-cli pandas numpy

# Clone tutorial repository
git clone https://github.com/Xether-AI/tutorials.git
cd tutorials

# Set up API credentials
export XETHER_API_KEY="your-api-key"
```
